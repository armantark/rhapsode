# Product Context

Generic flashcards and first-letter memorization tools do not adequately train
continuous spoken recall. Rhapsode treats the passage, its hierarchy, and oral
performance as the primary learning objects.

The core loop is: prepare a passage, learn small recall targets, chain them,
perform delayed spoken retrieval, self-grade the attempt, and revisit weak links
at useful intervals. Linguistic annotations support recall without becoming the
learning objective.
