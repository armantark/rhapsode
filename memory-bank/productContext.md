# Product Context

Generic flashcards and first-letter memorization tools do not adequately train
continuous spoken recall. Rhapsode treats the passage, its hierarchy, and oral
performance as the primary learning objects.

The core loop is: prepare a passage, learn small recall targets, chain them,
perform delayed spoken retrieval, self-grade the attempt, and revisit weak links
at useful intervals. Linguistic annotations support recall without becoming the
learning objective.

## Primary User

A single user with a CS+linguistics background memorizing long epic verse
(currently the Iliad) and liturgical texts (Classical Armenian). Design
implications we committed to:

- Attention is the scarce resource: sessions must be short and winnable, so
  the smart planner caps sessions at 12 items and triages (weak links →
  learning → new → maintenance).
- Motivation is deadline-driven: the due-count banner and one-click
  "practice due" buttons act as the deadline surrogate.
- One-click start beats configuration: smart sessions are the default; manual
  mode selection is behind a disclosure.
