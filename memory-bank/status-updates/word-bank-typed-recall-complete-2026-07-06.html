typed_recall QA: PASS

Gold lead-in cue, textarea, then the attempt survives the reveal — stacked directly above the true line in the passage font for the visual check. Works for lines and as a typed juncture bridge ("type just the next line's first N words"). Number keys can't grade while typing (input focus is excluded), and Space types a space instead of revealing.

What changed where

  • PracticeMode enum gained both modes; OpenAPI contract and the TypeScript client regenerated; the manual launcher lists both buttons.
  • Rotation: learning = word_bank → cue_recall → chaining → fading; graduated = random_start → typed_recall → chaining → cue_recall; graduated junctures add typed_recall third. Latency defaults: 40s / 60s until personal history takes over.
  • PromptCard got both renderers; per-item state reset is now guarded on the item id so a reveal can never wipe an in-progress draft or arrangement.
  • Smart-rotation e2e updated: a learning line's second smart session now deals word bank before cue recall.
